# Ruby_linked_list

This project is to create a linked list class and its methods. The class itself is in `lib/linked_list`. File `main.rb` is to test my linked list. All were written in Ruby.



## Public Methods
* #append(value)
* #prepend(value)
* #size
* #head
* #tail
* at(index)
* #pop
* #contains?(value)
* #find(value)
* #to_s
* #insert_at(index)
* #remove_at
